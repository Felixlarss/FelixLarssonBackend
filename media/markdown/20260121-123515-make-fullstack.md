# 20260121-123515-make-fullstack

# backend

```bash
npm create vite@latest
```

# fromntend

```bash
npm init -y

npm install express cors dotenv pg

#install types
npm i --save-dev @types/pg

npm i --save-dev @types/node

npm i --save-dev @types/express

npm i --save-dev @types/cors

#make essential files
touch .env

touch server.js

touch database.ts
```

## In .env

```dotenv
PGURI='postgresql://username:password@localhost:3000/postgres'

#In this case
PGURI='postgresql://postgres:secret@localhost:3000/postgres'

```

## In server.js

```typescript
import express from "express";
import cors from "cors";

const app = express();
const PORT = 3000;

app.use(express.json());

app.use(cors());

app.listen(PORT, () => {
  console.log("Listening on port " + PORT);
});
```

## In database.ts

```typescript
import "dotenv";
import { Client } from "pg";

export const db = new Client({
  connectionString: process.env.PGURI!,
  ssl: {
    rejectUnauthorized: false,
  },
});

db.connect();
```
